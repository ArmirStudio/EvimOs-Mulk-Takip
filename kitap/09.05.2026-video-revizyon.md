## Doğrulama

Çalıştırılan komutlar:

```powershell
python -m compileall backend
npm.cmd run lint
npx.cmd tsc --noEmit --pretty false
```

Sonuç:

- Backend compile başarılı.
- Frontend lint başarılı, error yok.
- Lint warning sayısı 12.
- `tsc --noEmit` proje genelindeki eski navigation/brand tipi uyuşmazlıkları nedeniyle başarısız oluyor; bu komut package script'i değildir. Bu revizyonda dokunulan shared screen dosyalarında filtrelenen yeni hata görülmedi.
