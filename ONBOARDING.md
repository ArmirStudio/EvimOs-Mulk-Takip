### MCP Servers to Activate
- [ ] Claude in Chrome — Lets Claude control and read your Chrome browser (navigate pages, fill forms, read page content, capture screenshots). Install the Claude in Chrome extension from the Chrome Web Store and enable it in your Claude Code settings.

### Skills to Know About
- `/model` — Switch between Claude models mid-session. Used frequently here to balance speed vs. capability on different task types (e.g. Haiku for quick lookups, Opus for complex architecture).
- `/compact` — Compress conversation history to save context when sessions run long. Use it when Claude starts feeling slow or repetitive on a long debugging session.
