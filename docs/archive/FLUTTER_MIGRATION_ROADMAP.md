# Flutter Migration Roadmap

Bu dosya tarihsel referanstir. Guncel urun plani React Native Expo kod tabani uzerinden devam eder.

## Durum
- Flutter yeniden yazim plani aktif backlog'da degildir.
- Bu belge yalnizca eski bir alternatif teknik yonu temsil eder.
- Canli kararlar icin `docs/state.md` ve `docs/phases.md` kullanilmalidir.

## Not
- Flutter gecisi yeniden degerlendirilecekse yeni plan sifirdan yazilmalidir.
- Bu dosya canli mimariyi veya mevcut release hedeflerini temsil etmez.
